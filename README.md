# Android Chatting Application

A feature-rich chatting application built with **Java** in **Android Studio** using **Firebase** for backend data storage and real-time messaging. The app demonstrates mobile development, user authentication, real-time database integration, and multimedia messaging.

## Features
- User registration and login via Firebase Authentication
- Add friends or followers
- Send text messages in real-time
- Send and receive images
- Change profile pictures
- View online/offline status
- Push notifications for new messages (optional)
- Clean and responsive UI using Android layouts

---

## Technologies
- **Programming Language:** Java  
- **IDE:** Android Studio  
- **Backend:** Firebase Realtime Database, Firebase Authentication, Firebase Storage  
- **UI:** XML Layouts, Material Design components  
- **Other Tools:** Glide (for image loading), Firebase Cloud Messaging (optional)
